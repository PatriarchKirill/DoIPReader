# DoIP диагностический клиент

Клиент для удалённой диагностики автомобилей по протоколу DoIP (Diagnostic over IP, ISO 13400).  
Позволяет подключаться к ЭБУ, отправлять UDS-запросы и расшифровывать ответы.

## Основные возможности

✅ Подключение к автомобилю по TCP (стандартный порт 13400)  
✅ Отправка диагностических запросов (например, чтение данных по идентификатору 0xF190)  
✅ Приём и расшифровка ответов:
   - VIN номер
   - Логический адрес ECU
   - EID (Entity ID)
   - GID (Group ID)
✅ Интерактивный режим с вводом сообщений в hex  
✅ Обработка таймаутов и ошибок соединения

## Быстрый старт

1. Убедитесь, что компьютер и автомобиль находятся в одной сети (например, через Ethernet).
2. Установите IP автомобиля в переменной `DOIP_IP`.
3. Запустите скрипт:

```bash
python main.py
