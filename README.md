# 🛠 SVI TOOL

**Программа для диагностики автомобилей Zeekr 001** 🚗

---

## 📌 О программе

SVI TOOL подключается к автомобилю через сеть (DoIP) и позволяет:

- 🔍 Смотреть ошибки (DTC)
- 🧹 Очищать ошибки
- 📲 Обновлять прошивку
- 👨‍💻 Работать с разработчиками

## ⚠️ Примечание

На данный момент **единственная рабочая вкладка — Diagnostic**. Остальные вкладки пока не реализованы.

---

## ⚙️ Как это работает

- 🌐 Программа подключается к автомобилю по IP **169.254.19.1**
- 🔌 Порт подключения: **13400**
- 📡 Общается через протокол **DoIP** (стандарт ISO 13400)

---

## 📂 Логи

Все действия сохраняются в папке `bin/cash/cash_diag/`

- 📄 `cash_diag.txt` — лог диагностики
- 📊 `connection_status.json` — статус подключения и VIN

---

## 💻 Требования

- 🐍 Python 3
- 🪟 Windows или 🐧 Linux
- 🌍 Сеть (Ethernet) до автомобиля
