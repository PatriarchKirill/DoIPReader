# 🛠 SVI TOOL v2.0

**Модульная диагностическая программа для автомобилей Zeekr 001** 🚗

---

## 📌 О программе

SVI TOOL v2.0 подключается к автомобилю через сеть (DoIP) и позволяет:

- 🔍 Смотреть ошибки (DTC)
- 🧹 Очищать ошибки
- 📲 Обновлять прошивку
- 👨‍💻 Работать с разработчиками

---

## ⚠️ Важно

> На данный момент **единственная рабочая вкладка — Diagnostic** ❗  
> Остальные вкладки (Workshop, Developer, ADB, Upgrade OS, Profile) пока **не реализованы** и находятся в разработке. 🚧

---

## 🏗 Что нового во второй версии

- 🧩 **Модульная архитектура** (config, logger, doip_client)
- 📝 **Полноценное логирование** с ротацией по дням
- 🔄 **Асинхронные операции** (GUI не зависает)
- 🖥 **Выделенный DoIP клиент** с reconnect и keep-alive
- 📁 **Централизованная конфигурация** (config.py)

---

## ⚙️ Как это работает

- 🌐 Подключение по IP **169.254.19.1**
- 🔌 Порт **13400**
- 📡 Протокол **DoIP** (ISO 13400) + **UDS**

---
